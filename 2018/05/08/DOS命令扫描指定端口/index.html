# DOS 命令扫描指定端口并查看端口号

# 使用命令 netstat -aon|findstr "80"

此命令可以查看 80 端口的占用情况


其中可以得知占用 80 端口的任务 PID 号为 4

# 根据 PID 号找出具体任务：tasklist|findstr "4"


可以看到其服务为 System。
