打开防火墙8090端口

iptables -I INPUT -p tcp -m state –state NEW -m tcp –dport 8090 -j ACCEPT
service iptables save
service iptables restart #打开防火墙8090端口

关闭端口号:
iptables -A OUTPUT -p tcp –dport 8090 -j DROP
打开端口号：
iptables -A INPUT -ptcp –dport 8090 -j ACCEPT
iptables -I INPUT -ptcp –dport 8090 -j ACCEPT

测试端口是否打开
telnet 69.171.68.121 8090
nc -z 69.171.68.121 8090
telnet localhost 8090

多方网页测试，证明8090端口其实不可用的，使用命令netstat -tnl不能扫描出来。使用telnet 69.171.68.121 8090命令没有任何反应，即使使用127.0.0.1和localhost也是如此。而且8000端口则可以任意连接(断开连接方式ctrl+]，然后进入了telnet界面后使用quit退出)。
