# 被裁时：哪些文件**绝对不能签**，哪些**必须签**

### 一、❌ 绝对不能签的文件
| 文件名称/内容 | 为什么不能签 |
|---------------|--------------|
| 辞职申请书 | 一签＝主动离职，**无赔偿** |
| 自愿离职申请 | 同上，放弃一切补偿 |
| 个人原因离职 | 等于你自己要走，被裁变辞职 |
| 协商一致无任何纠纷 | 没拿到钱/没谈好，千万别签 |
| 空白文件、空白纸 | 后期随便填内容，你说不清 |
| 放弃经济补偿承诺书 | 直接放弃赔偿 |

---

### 二、✅ 可以/应该签的文件
| 文件名称 | 注意点 |
|----------|--------|
| 解除劳动合同协议书 | 必须写清：**赔偿金额、发放时间、离职日期** |
| 工作交接单 | 交接清楚，避免背锅 |
| 离职证明 | 必须开，**不能写负面评价** |
| 工资/补偿金结算单 | 确认金额无误再签 |

# 被裁员「三不说」&「三不签」核心总结

#### ✅ 三不说（沟通避坑）
| 不说内容 | 核心原因 |
|----------|----------|
| 不主动说「因个人原因离职」 | 这是放弃补偿的“自杀式”表述，所有沟通都要坚持是**公司提出解除** |
| 不认可协议中「双方无任何争议」条款 | 这条会让你放弃加班费、未休年假工资等所有未结款项 |
| 不透露个人急迫情况和底线 | 别说“房贷压力大”“急着找工作”，会暴露软肋，让对方在赔偿上压价 |

---

#### ✅ 三不签（文件避坑）
| 文件类型 | 签字要求/注意事项 |
|----------|------------------|
| 《协商解除协议》 | **必须写明**：赔偿金额（精确到元）、支付日期（具体到日）、离职原因（务必是「协商一致解除」），否则不签 |
| 《解除劳动合同通知书》 | 若公司单方面出具且理由不合理，收到后**手写注明**：「本人对解除理由不认可，保留申请劳动仲裁的权利」，并拍照留证，不要直接签字认可 |
| 《自愿离职申请书》 | **绝对不能签**！一旦签字，就从「公司裁员」变成「主动辞职」，N/N+1 经济补偿金一分钱都拿不到 |

---

💡 一句话提醒：
- 所有签字前，先确认**离职原因是公司提出/协商解除**，且**赔偿金额和支付时间写死**，否则一律拒签。
- 任何文件都要先拍照留存，再考虑是否签字。
