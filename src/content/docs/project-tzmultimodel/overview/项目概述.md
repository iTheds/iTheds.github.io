## 项目定位

`TzMultiModel` 是基于 Qt 的多模数据库管理客户端，目标是统一关系型与时序型数据库的图形化管理能力，覆盖连接管理、SQL 执行、导入导出、权限管理、数据回放与校验等场景。

## 功能范围

- 数据库连接与会话管理
- SQL 执行与结果展示
- 元数据浏览与对象管理
- 数据导入导出与备份恢复
- 用户与授权管理
- 主题订阅与数据同步

## 模块组成

- 表现层：`view/`
- 控制层：`controller/`
- 中间件层：`model/middleware/`
- 驱动与接入层：`tools/odbc`、`tools/odbc-jni`、`tools/TZMjdbc`
- 服务与部署：`server/`、安装脚本

## 项目特征

- 不是单一 GUI，而是“客户端 + 中间件 + 驱动链路 + 测试”一体化工程。
- 具备跨语言接入能力，支持 ODBC/JDBC 生态联通。
