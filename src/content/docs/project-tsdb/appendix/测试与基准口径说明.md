## 目录口径

- `tests/sqllogictest`：SQL 逻辑回归测试。
- `tests/server_tests`：服务与集群相关测试。
- `tests/disk_test`：磁盘读写性能测试。
- `benchmarks`：TPC-H 等基准流程说明（按需启用）。

## 使用建议

- 功能正确性优先使用 `sqllogictest + server_tests`。
- 性能评估优先使用 `disk_test + benchmarks`，并记录硬件与配置环境。
