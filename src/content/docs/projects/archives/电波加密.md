# 电波加密解密

> 原理

将morse电码与音频的波长相对应，来进行加密过程。
解密只需要将音频文件分析一下，得出morse电码加密。
倘若还有加密的过程，可能需要在栅栏密码中进行分析，使用CTFcrack来分析栅栏密码，就可以得到flag。

> 工具

* Audacity音频编辑软件
* [morse在线解密网站](http://rumkin.com/tools/cipher/morse.php)
* [CTFcrack框架](https://github.com/0Chencc/CTFCrackTools)
