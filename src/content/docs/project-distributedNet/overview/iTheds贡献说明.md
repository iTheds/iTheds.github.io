## 责任范围

核心模块独立负责，覆盖需求调研、架构设计、实现、测试与演示联调。

## 主要贡献

- 架构设计：设计 `CONNECT + IOMode + NetPool` 分层通信架构，统一 TCP/UDP 接入并支持单播、组播、广播。
- 协议与可靠性：主导 UDP 分包、ACK 回执、超时重发与失败判定机制，并与链路状态检测联动。
- 拓扑与路由：实现节点可达状态管理与拓扑动态更新，支持多跳通信策略验证。
- 调度与自适应：构建多级多类任务队列，基于带宽与网络质量动态调整压缩和裁剪策略。
- 可观测性：补充链路监控与任务可视化，展示带宽、压缩算法、发送失败统计与节点上下线状态。
- 工程验证：组织多节点仿真与压测，覆盖 8MB/s、2MB/s、500KB/s、200KB/s 场景。
- 交付支撑：完成 Docker 化运行、外场联调、测试大纲与验收材料准备。
