# 文档projectdistributedNet

2024.09 - 2024.12

项目说明:面向弱带宽与不稳定链路环境的协同通信系统，围绕“可达状态管理 + 自适应分发 + 缓存重传”构建多节点可靠共享能力。
贡献共享度(职责口径):核心模块独立负责(需求调研、架构设计、实现、测试、演示)。

个人职责与成果:
- 架构设计:设计 `CONNECT + IOMode + NetPool` 分层通信架构，统一 TCP/UDP 接入方式，支持单播/组播/广播与任务驱动分发。
- 协议与可靠性:主导 UDP 分包与重传机制设计(包标记、ACK 回执、超时重发、失败判定)，并与链路状态检测联动。
- 拓扑与路由:实现节点可达状态管理(1 跳/2 跳/不可达)与拓扑动态更新机制，支持多跳通信策略验证。
- 自适应调度:实现多级多类任务队列，基于带宽与网络质量动态调整降采样阈值、压缩策略与任务调度优先级。
- 可观测性:完成链路监控与任务可视化能力，展示带宽、压缩算法、原始/压缩后数据长度、发送失败统计与节点上下线状态。
- 工程化测试:构建多节点仿真与压测环境，覆盖 8MB/s、2MB/s、500KB/s、200KB/s 带宽场景，评估丢包率与带宽利用率。
- 交付保障:完成 Docker 化运行与外场联调支持，补齐测试大纲、测试报告与验收材料，支撑项目指标验收流程。
