- 网络与分布式通信:net_pool、discovery、udp 相关实现和重构，处理 leader 收敛、启动并发、回退算法、网络读写接口统一。

- 测试体系建设:新增/重构大量单测、集成测试、回归测试和交互测试；覆盖 PG server、ODBC、发现机制、IPC 报告等场景。
