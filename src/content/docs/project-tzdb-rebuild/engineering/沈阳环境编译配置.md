## 配置概览

该配置用于沈阳现场环境下的 TZDB 编译与工具链适配，原稿主要包含编译宏、头文件搜索路径以及一个编译器兼容选项。

## 编译宏

| 宏或变量 | 说明 |
| --- | --- |
| `CPU=ARMV8A` | 目标 CPU 架构为 ARMv8-A |
| `ACOREMCOS_MCORE` | 启用多核相关宏 |
| `ACOREMCOS_64BIT` | 启用 64 位目标环境 |
| `BOARD=$(CONFIG_BOARD)` | 板级配置由外部变量注入 |
| `ACOREOS_CPP` | 启用 ACoreOS C++ 编译环境 |

## 头文件搜索路径

| 路径 | 用途 |
| --- | --- |
| `$(PLATFORM)/target/common/include/rtl` | 运行时基础头文件 |
| `$(PLATFORM)/target/common/include/rtl/sys` | 系统级运行时头文件 |
| `$(PLATFORM)/target/acoreosmp/include/posix` | POSIX 兼容层 |
| `$(PLATFORM)/target/acoreosmp/include` | ACoreOS MP 头文件 |
| `$(PLATFORM)/target/common/include` | 平台公共头文件 |
| `$(PLATFORM)/target/common/include/os` | OS 相关头文件 |
| `$(PLATFORM)/target/common/include/drv` | 驱动头文件 |
| `$(PLATFORM)/target/common/include/arch` | 架构相关头文件 |
| `$(PLATFORM)/target/common/include/inet` | 网络协议相关头文件 |
| `$(PLATFORM)/target/common/include/inet/sys` | 网络系统接口头文件 |
| `"${workspace_loc:/${ProjName}/src/inc}"` | 工程私有 `inc` 目录 |
| `"${workspace_loc:/${ProjName}/src/include}"` | 工程公共 `include` 目录 |
| `"${workspace_loc:/${ProjName}/src/tools/tzdb-odbc/inc}"` | ODBC 工具头文件 |
| `"${workspace_loc:/${ProjName}/src/tools/tzdb_fs/inc}"` | 文件系统工具头文件 |

## 编译兼容项

| 项目 | 说明 |
| --- | --- |
| `-fno-use-cxa-atexit` | 规避 `dso_handle undefined` 问题 |
