# Overflow 机制优化进度文档

> 关联设计文档:[overflow_optimization_design.md](溢出优化设计.md)

## 概述

优化 `Write` 和 `Update` 的 overflow 触发条件，从固定 64 字节阈值改为基于页面可用空间的动态阈值，与 `WriteRaw` 保持一致。
