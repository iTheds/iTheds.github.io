# TZDB Project Overview

TZDB 是一套面向嵌入式与服务化场景的数据库工程，核心覆盖：

- 数据库内核（存储、事务、元数据、索引）
- SQL 引擎（主链路 + 新链路并行演进）
- 远程访问服务（协议、网络、客户端 API）
- 集群高可用（主从、日志同步、恢复）
- 工具与生态（ODBC、命令行工具、联调工具）

## 工程目标

1. 高性能、可裁剪的嵌入式数据库能力
2. 面向网络访问的服务化能力
3. 主从模式的高可用与一致性保障
4. 跨平台构建、测试与交付体系
