# Distributed Database Performance Analysis 601

该分析整理了 601 阶段对数据库分布式链路的性能测试目标与结论：

- 关注单条消息增删改查（落盘/内存）性能。
- 关注网络同步 10000 包规模下不同报文尺寸性能。
- 数据范围覆盖 16B 到 64KB，含极值和阶梯值。

主要结论：

- UDP 在跨机场景下可用，但尾延迟控制仍需优化。
- 网络链路波动会放大数据库端表现差异。
- 单看平均值不足以评估稳定性，需联合极值和标准差。
