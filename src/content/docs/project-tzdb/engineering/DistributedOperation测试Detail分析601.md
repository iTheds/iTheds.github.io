# Distributed Operation Test Detail Analysis 601

基于 `601测试留存` 的分布式操作测试记录，重点关注 UDP 场景下不同报文大小(16B-64KB)的回传耗时、链路抖动与异常值。

关键现象：

- 小包到中包整体可测，但在天脉非单机场景抖动显著。
- 大包(接近 64KB)在部分轮次出现明显异常值。
- 单机参照测试与跨机测试存在数量级差异，说明网络拓扑与环境因素影响较大。

建议：

1. 将异常值样本和时钟源一起记录，避免误读吞吐结果。
2. 拆分“发送耗时”和“回传耗时”指标，避免链路混叠。
3. 固化天脉单机/跨机两套标准基线作为回归门槛。
