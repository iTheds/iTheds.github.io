| 接口名称                         | 功能描述                 | 异常表现 | 可能原因                   |
| -------------------------------- | ------------------------ | -------- | -------------------------- |
| `BoderAreaType_a12NodeCoord_at`  | 访问边界区域节点坐标数组 | 偶发异常 |  |
| `NodeCoordType_Long_put`         | 设置节点经度值           | 偶发异常 |    |
| `NodeCoordType_Lat_put`          | 设置节点纬度值           | 偶发异常 |    |
| `BoderAreaType_a12NodeCoord_put` | 更新边界区域节点坐标     | 偶发异常 | 内存分配失败 |

输入的都是新的数据，并非 update 。
