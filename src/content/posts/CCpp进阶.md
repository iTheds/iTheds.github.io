## 基本思想

C++是面向对象的程序设计方法。
定义？其实OPP(Object Oriented Programming)就是一种模块化的思想。
专业的定义是将现实的东西进行抽象化描述为对象并建立模型。本质上是分类并集合，塑造并构建。

我们可以大致想象一下编程的过程，写很大(取极限)的程序的时候，头文件一堆，函数也是一堆的，只学了C语言的基础的时候，用一个数组在描述海量的东西，二维数组就在存放着各种自定义的数据，它就可以被解释为一个对象，联合头文件中所存有的方法，就可以认为是一种class。
基于此，然后就诞生了衍生出更加快捷，即便于理解，又符合逻辑效率的，面向对象的编程方法。

## 继承与派生

软件重用性的体现。

一个新类冲已有的类那里获得其已有的特性，这种现象称为类的继承(inheritance)；从一个已有的类(父类)中产生一个新的子类，称为类的派生。

派生类的基类的具体化，基类是派生类的抽象。

class 派生类名: [继承方式] 基类名
{
    派生类新增加的成员;
}

继承方式包括：
1. public：基类的公有成员和保护成员在派生类中保持原有访问属性，其私有成员仍为基类私有。
2. private：基类的公有成员和保护成员在派生类中成了私有成员，其私有成员仍为私有。
3. protected：基类的公有成员和保护成员在派生类中成了保护成员，私有成员仍未基类私有。

保护成员的意思是，不能被外界引用，但可以被派生类的成员引用。

* 派生与析构函数

    派生类中执行析构函数时，一般先调用派生类析构函数再调用基类析构函数。
    但如果是用new运算符建立临时对象，并定义了一个指向基类的指针变量。在程序中用带指针参数的delete撤销对象时，系统只会执行基类的析构函数。
    如果要执行派生类的析构函数，需要声明基类析构函数为虚函数。

        Point *p = new Circle;


## 多态与虚函数

多态就是不同的集合对相同的指令做出不同的回应。
专业术语：向不同的对象发送同一个消息，不同的对象再接收时会产生不同的行为(即方法)。
C++的虚函数就是用来解决动态多态问题的。

在类中声明虚函数。其派生类中的同名函数都自动称为虚函数，派生类中声明可以加也可以不加virtual，但是为了方便辨识统一加上。

    virtual void display();

虚函数原理基于：
1. 我们可以通过父类定义的指针去指向子类。
2. 派生类重新定义父类同名函数，仍然执行父类同名函数。

所以虚函数类似于JAVA中的重写功能。

### 纯虚函数与抽象类

纯虚函数：

    virtual void shapeName() const = 0;

只有虚函数的类，为多态性服务的类称为抽象类。

## 指针

为什么要一上来就放指针？因为指针和OOP编程思想才是C++的关键，其他的一概其次。

指针是基于存储数据时有一个地址。
地址指向该变量单元。因此将地址形象化为指针。通过指针就可访问以它为地址的内存单元。一个变量的地址称为该变量的指针。
一个变量准们用来存放地址(即指针)的，则称它为指针变量。

定义

    int *i_pointer;

用int定义的指针变量只能存放int类型的数据的地址。

    int i;
    i_pointer = &i;

(1) `&*pointer_1`表示：指针pointer_1指向的变量的地址。
(2) `*&a`表示：等价于变量a。

&为取地址运算符，将之后的变量地址取出；而*则是之后变量(解释为地址)所指向的变量。

用指针作为函数参数，在函数内改变指针指向的变量的值，那么这里面的值将会改变。
换而言之，使用*pointer时会改变其数值，但是使用pointer时不会改变，因为其作为形参的本质没有改变。

```C++
//正确做法：改变了指向的变量的值
void swap(int *p1,int *p2){
    int temp;
    temp = *p1;
    *p1 = *p2;
    *p2 = temp;
}
//只改变了指针的指向，但是由于函数形参的特性，没有动摇原形参的指向
void swap(int *p1,int *p2){
    int *temp;
    temp = p1;
    p1 = p2;
    p2 = temp;
}
```

指针在指向数组或者字符串的时候，通常可以使用p+i的方式进行对数组或者字符串中的数据进行指向。

`const int *p;`const指针，不能通过指针p来改变指针所指向的变量的值，但是可以改变指针的指向。
`char * const p;` 表示指针指向的变量是不能改变的。
`const int * const p;` 表示指向不能改变并且指向的变量值也不能改变。

### 数据的存储

内存中供用户使用的存储空间分3部分：
1. 程序区
2. 静态存储区
3. 动态存储区

存储期storage duration，分为静态存储期和动态存储期。
存储方法分为静态存储和动态存储两大类。
同时存储类别也分为4种：自动的(auto)、静态的(static)、寄存器的(register)、外部的(extern)。

Auto：
自动分配内存。属于动态局部变量，函数调用结束后即释放。编译后如果没有赋值是一个不确定的值。

Static：
指定为静态局部变量。在静态存储区内分配存储单元。在整个运行期间都不释放。编译后如果没有赋值是0或者空字符。
每次调用该函数，该变量保存上一次使用的值。

Register：
Auto和Static都是存放在内存中的，使用的时候由控制器发出指令进行调用到运算器。对于使用频繁的变量使用Register放在寄存器中，可以提高执行效率。

Extern：
声明全局变量。与Static的区别是，static是只能用域本文件的外部变量(全局变量)称为静态外部变量。。

而Extern和Static也可以用于函数。分外部函数和内部函数。

### 数组

返回数组函数是怎么样的呢，首先数组中的元素作为参数传递的时候与普通的数据是一样的，形式参数是无法改变的，整个函数作为参数传递的时候是可以改变的，因为本质上是传递的地址。

    int max_value(int arrray[][])
    int max_value(int arrray[2][])

当数组是字符串数组时，可以如下直接输入输出：

    char str[20];
    cin>>str;
    cout << str;

但是要注意，{'1','a','v'}和 "1av"还是有'\0'的区别的。

关于返回数组函数：C++ 不允许返回一个完整的数组作为函数的参数。但是，可以通过指定不带索引的数组名来返回一个指向数组的指针。

```C++
#include <iostream>
#include <cstdlib>
#include <ctime>
 
using namespace std;
 
// 要生成和返回随机数的函数
int * getRandom( )
{
  static int  r[10];
 
  // 设置种子
  srand( (unsigned)time( NULL ) );
  for (int i = 0; i < 10; ++i)
  {
    r[i] = rand();
    cout << r[i] << endl;
  }
 
  return r;
}
 
// 要调用上面定义函数的主函数
int main ()
{
   // 一个指向整数的指针
   int *p;
 
   p = getRandom();
   for ( int i = 0; i < 10; i++ )
   {
       cout << "*(p + " << i << ") : ";
       cout << *(p + i) << endl;
   }
 
   return 0;
}
```

|setprecision(n) | 设置实数的精度为n位。在以一般十进制小数形式输出时，n代表有效数字。在以fixed(固定小数位数)形式和scientific(指数)形式输出时，n为小数位数。 
|setw(n) | 设置字段宽度为n位。 
|setiosflags(ios::fixed) | 设置浮点数以固定的小数位数显示。 
|setiosflags(ios::scientific) | 设置浮点数以科学计数法(即指数形式)显示。 
|setiosflags(ios::left) | 输出数据左对齐。 
|setiosflags(ios::right) | 输出数据右对齐。 
|setiosflags(ios::shipws) | 忽略前导的空格。 
|setiosflags(ios::uppercase) | 在以科学计数法输出E和十六进制输出字母X时，以大写表示。 
|setiosflags(ios::showpos) | 输出正数时，给出“+”号。 
|resetiosflags | 终止已设置的输出格式状态，在括号中应指定内容。
