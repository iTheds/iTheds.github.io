### 重建二叉树 输入某二叉树的前序遍历和中序遍历的结果，请重建出该二叉树。假设输入的前序遍历和中序遍历的结果中都不含重复的数字。例如输入前序遍历序列{1,2,4,7,3,5,6,8}和中序遍历序列{4,7,2,1,5,3,8,6}，则重建二叉树并返回。

前序和中序遍历，指的是根节点在前和中。
所以主要的思想是截断，分块进行。

前序遍历中第一位 1，是根节点，中序遍历分块：{4,7,2},`{1}`,{5,3,8,6}。
然后再看前序{2,4,7,3,5,6,8}，是2，
left分{4,7},`{2}` -> `{4}`,{7}。
right分{5},`{3}`,{8,6} -> {8},`{6}`。
所以确定要用到递归。

```C++
class Solution
{
public:
    void push(int node) {
        stack1.push(node);
    }

    int pop() {
        int tmp = 0;
        if(!stack2.empty()){
            tmp = stack2.top();
            stack2.pop();
            return tmp;
        }else if(stack1.empty()){
            return NULL;
        }else{
            while(!stack1.empty()){
                stack2.push(stack1.top());
                stack1.pop();
            }
            tmp = stack2.top();
            stack2.pop();
            return tmp;
        }
    }
```

```php
    while (!$stack->isEmpty()) {
         $stack2->push($stack->pop());
    }
    $dst = $stack2->pop();
    $stack2->rewind();

         $stack->push($stack2->pop());
    }
    return $dst;
}
```

### 跳台阶 一只青蛙一次可以跳上1级台阶，也可以跳上2级。求该青蛙跳上一个n级的台阶总共有多少种跳法（先后次序不同算不同的结果）。

两个解决方法，第一种遍历递归走一遍，第二种用理论知识数学计数原理。
看一下第一种解法，记忆并递归，走一遍所有可能性，绝对不会错，用count(j)进行计数。一开始是判断了 `i > number` 和 `i ==number`，那么就会花费多有的资源进行无用的计算递归`i=number+1`的情况，所以直接在`number-1`的时候停止。
网上比较快的方式是动态规划，分成子问题，比如到n级的方法可以是基于n-2的，而n-2级的是基于n-4的。
所以n级的跳法$f(n)=f(n-1)×1+f(n-2)×1$，为什么f(n-2)也是`1`种情况呢，因为f(n-1)种包含了f(n-2)到f(n)的跳1级方法。
以此类推，那么可以得知这是一个斐波那契数列。第一种方法的改良v2.0如代码v2。

其实不需要用到动态规划的思想也可以发现是斐波那契数列，那么就是自上而下，从n开始往下走，这个时候就可以发现尾部数值都是1，所以在末尾的时候累加1也是可以的，这样就不需要一个`i`来进行存储进度了。

排序 $A\frac{m}{n}$ ，组合的思想是 $C\frac{m}{n}=\frac{A\frac{m}{n}}{A\frac{m}{m}}$，组合按照排序的方法取出，之后除以按照取出的数的排序，从而锐减到组合的次序。

插板子的思想，次数x可以取值的范围是(n为奇数)$[(n-1)/2,n-1]$;
当 $x = (n-1)/2$时，只有一种可能就是全部都是1级跳，跳法 +1;
当 $x = (n-1)/2 +1$时，只存在1个2级跳，n-1个坑里面选一个不插板子，跳法 +(n-1);
当 $x = (n-1)/2 +2$时，存在2个2级跳，n-1个坑里面有2个不插板子，并且不可以相邻，跳法 + !(n-1-2);
当 $x = (n-1)/2 +3$时，存在3个2级跳，n-1个坑里面有3个不插板子，并且不可以相邻，跳法 + (n-1-1)(n-1-3)(n-1-6)/!3;
当 $x = (n-1)/2 +m$时，存在m个2级跳，n-1个坑里面有m个不插板子，并且不可以相邻，跳法 + ?;

问题是怎么处理相邻的位置，有两种情况。

### 输入一个整数，输出该数32位二进制表示中1的个数。其中负数用补码表示。

模2 并移位 ， 移位异或 都是32次。

val 和 (val - 1) 进行与操作。赋值给val，直到val为0。
空间复杂度直接就是O(n)，n为1的个数。

怎么理解？
val - 1.
10000 - 1 = 1111
10001 - 1 = 10000

一种理解是，利用与的特性。反正val 和 任何数取与，得到的数中包含的 1 ，都不会超过val。那不如就是找一个方法，消除val中的1。并且消除1的个数还是一个一个消除的，那么就得到了本题结果。而一个一个消除的方法，就是取d = 1，这样就跳过了一段0很多的情况。关于d取其他数，可行性未知，有可能会适得其反。

这是逆向思维，但是顺势思维未有多想。

### 给定一个double类型的浮点数base和int类型的整数exponent。求base的exponent次方。

保证base和exponent不同时为0。不得使用库函数，同时不需要考虑大数问题，也不用考虑小数点后面0的位数。

需要考虑正负问题。
正直接乘。
判断正负，如果是负数，1/base。

下例程序：
1. 逗号运算符，使得if语句不使用{}而执行多次操作。
2. 移位运算
3. 适应移位运算的res。

```C++
class Solution {
public:
    double Power(double base, int exponent)
    {
        double res = 1.0;
        if(exponent<0)base = 1.0/base,exponent*=-1;
        while(exponent)
        {
            if(exponent&1)res = res*base;
            base*= base;
            exponent>>=1;
        }
        return res;
    }
};
```
