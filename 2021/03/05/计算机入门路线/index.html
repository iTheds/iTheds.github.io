# 计算机入门路线

这里提供速学的路线，目的是能够开发基本的网页。
附赠开发微信小程序和 app 的大概教程，因为网页我还是比较熟悉，但是后面二者不太熟悉。
当然以上不代表能够入门到计算机的知识体系中，只是能够上手进行应用层的设计和实用。

# 制作网页

软件工具就使用 Visual Studio Code (简称 VS Code)，下载安装即可 (注意，不是 Visual Studio)。

  • 前端
    必备技能：html，css，javascript (简称 js)。
    说明：其实如果是最简单的静态网页的话那么这个就完全是够了。不涉及到数据库方面，也不需要自己部署到服务器，那么用以上的语言设计完之后就可以交付了。
    增值技能：React.js,vue.js。其实都是 js 的框架，个人 React 比较熟悉，学习其中一个就可以。框架可以让你的设计更加的方便，页面更美观，框架可以帮助你解析。
  • 服务器端 (如果使用 github，则后续可不学)：
    必备技能：linux 命令，http 协议。
    说明：linux 命令的话，需要用即刻搜就可以。http 协议的实现可以使用 apache 环境或者 nodejs 环境。http 协议超文本传输协议，其实 html,css,js 都是基于这个环境的。
    增值技能：SQL 语言，Linux 命令的进阶 (实际就是学习各种 linux 的底层包括但不限于权限控制，进程线程等)，还有许多的可以学习。
  • 后端 (不用看)：
    必备技能：一门后端语言，可以是 java，也可以是 php (这个我熟)，或者 Golang，python，听说 Asp.net (C#) 也可以。
    说明：就是写一些 API，调用数据库的数据，进行分析，从服务端传输到网页端使用 JS 进行接收和解析。其实 php 也可以写前端，还有 java + vue 的方式 web 开发，Golang 和 python 则是待了解了。
    增值技能：这个大可不必了。基本的就是语言的强化，但是后端代表的是浩如烟海的知识内容，不仅要对操作系统、数据结构有了解，甚至还要对计算机组成原理、计算机网络进学习和实践。所以后端的学习如果执着于语言，那么会有禁锢的效果。

总结：前面一堆文字似乎很多，但是要完成网页其实就是 html,css,js。

这里涉及到一个 github，个人网站中有所提及。

# 微信小程序

微信小程序就是类似于 app 的开发方式，但是简单很多。
整个小程序框架系统分为两部分：逻辑层（App Service）和 视图层（View）。

参考教程 - 微信官方文档

个人没有开发过，但是我可以确定的是，看完官方文档中的指南加上百度搜索，那就可以进行自我的开发。
代码不懂怎么办，很简单，学一下 js 就可以大概看懂。然后整个开发也是组件的思想，模块化，需要什么在网上找并且调用就可以，然后更改参数。

我来说一下我的理解：
流程：注册微信公众号，下载软件 (微信小程序开发)，进行开发 (找模块并且调用，简单来说就是复制粘贴)，链接到微信公众号的开发者设置，发布。

其实可能就是搭建环境的时候还有发布的时候难一些。要做到开发非常简单的应用，只要三天即可。

# APP 开发

桌面应用用 C#，手机安卓端使用 Android。

着重说一下 Android，基本上就是在开发工具中进行写代码，有点类似于 JAVA 和 XML 语言，调用，设计。这块我懂得的不多，也没有太大的发言权。

然后后端和服务器端不变，制作网页篇的服务器和后端，大部分都适用。

# 计算机知识体系

此阙纯属瞎侃了。我的课程即是如此。先写这么多叭。

关于计算机的学习，如果只是一门工具，那么很快就可以做的很好，设计的也很好，甚至在目前的市场需求中可以跟学了很多年的人做的一样华丽。
但是如果深入到底层，涉及到算法、数学，追求效率和应对实时性、耦合性、健壮性，那么也许会有所不同吧。毕竟下笔千文，问起别人有没有吃饭还不是说一句，“你吃了吗”。

  1. 熟悉一门计算机基本语言，C 语言和 C++。

  2. 写一些简单的编程题目，可以上牛客或者力扣，ACM 练习网站也可以。

  3. 数据结构，操作系统，计算机组成原理，计算机网络。按顺序进行学习。最基本的即是这些。

  4. Linux 可以开始上手了。

  5. 软件开发，java 学习并且进行 web 端开发。接下来就是各种程序设计了。

  6. 硬件开发，学习嵌入式开发，单片机，无线传感器网络，通信原理。
