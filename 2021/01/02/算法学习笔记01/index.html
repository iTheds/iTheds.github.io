# 树和链表

# 重建二叉树 输入某二叉树的前序遍历和中序遍历的结果，请重建出该二叉树。假设输入的前序遍历和中序遍历的结果中都不含重复的数字。例如输入前序遍历序列 {1,2,4,7,3,5,6,8} 和中序遍历序列 {4,7,2,1,5,3,8,6}，则重建二叉树并返回。

前序和中序遍历，指的是根节点在前和中。
所以主要的思想是截断，分块进行。

前序遍历中第一位 1，是根节点，中序遍历分块：{4,7,2}, {1} ,{5,3,8,6}。
然后再看前序 {2,4,7,3,5,6,8}，是 2，
left 分 {4,7}, {2} -> {4} ,{7}。
right 分 {5}, {3} ,{8,6} -> {8}, {6} 。
所以确定要用到递归。

# 跳台阶 一只青蛙一次可以跳上 1 级台阶，也可以跳上 2 级。求该青蛙跳上一个 n 级的台阶总共有多少种跳法（先后次序不同算不同的结果）。

两个解决方法，第一种遍历递归走一遍，第二种用理论知识数学计数原理。
看一下第一种解法，记忆并递归，走一遍所有可能性，绝对不会错，用 count (j) 进行计数。一开始是判断了 i > number 和 i ==number ，那么就会花费多有的资源进行无用的计算递归 i=number+1 的情况，所以直接在 number-1 的时候停止。
网上比较快的方式是动态规划，分成子问题，比如到 n 级的方法可以是基于 n-2 的，而 n-2 级的是基于 n-4 的。
所以 n 级的跳法f(n)=f(n−1)×1+f(n−2)×1f(n)=f(n-1)×1+f(n-2)×1f(n)=f(n−1)×1+f(n−2)×1，为什么 f (n-2) 也是 1 种情况呢，因为 f (n-1) 种包含了 f (n-2) 到 f (n) 的跳 1 级方法。
以此类推，那么可以得知这是一个斐波那契数列。第一种方法的改良 v2.0 如代码 v2。

其实不需要用到动态规划的思想也可以发现是斐波那契数列，那么就是自上而下，从 n 开始往下走，这个时候就可以发现尾部数值都是 1，所以在末尾的时候累加 1 也是可以的，这样就不需要一个 i 来进行存储进度了。



根本理论就是排序和组合，排序是取出的数亦有顺序，组合是没有顺序；
排序 AmnA\frac{m}{n}Anm​ ，组合的思想是 Cmn=AmnAmmC\frac{m}{n}=\frac{A\frac{m}{n}}{A\frac{m}{m}}Cnm​=Amm​Anm​​，组合按照排序的方法取出，之后除以按照取出的数的排序，从而锐减到组合的次序。

插板子的思想，次数 x 可以取值的范围是 (n 为奇数)[(n−1)/2,n−1][(n-1)/2,n-1][(n−1)/2,n−1];
当 x=(n−1)/2x = (n-1)/2x=(n−1)/2 时，只有一种可能就是全部都是 1 级跳，跳法 +1;
当 x=(n−1)/2+1x = (n-1)/2 +1x=(n−1)/2+1 时，只存在 1 个 2 级跳，n-1 个坑里面选一个不插板子，跳法 +(n-1);
当 x=(n−1)/2+2x = (n-1)/2 +2x=(n−1)/2+2 时，存在 2 个 2 级跳，n-1 个坑里面有 2 个不插板子，并且不可以相邻，跳法 + !(n-1-2);
当 x=(n−1)/2+3x = (n-1)/2 +3x=(n−1)/2+3 时，存在 3 个 2 级跳，n-1 个坑里面有 3 个不插板子，并且不可以相邻，跳法 + (n-1-1)(n-1-3)(n-1-6)/!3;
当 x=(n−1)/2+mx = (n-1)/2 +mx=(n−1)/2+m 时，存在 m 个 2 级跳，n-1 个坑里面有 m 个不插板子，并且不可以相邻，跳法 + ?;

问题是怎么处理相邻的位置，有两种情况。

# 矩阵覆盖 我们可以用 21 的小矩形横着或者竖着去覆盖更大的矩形。请问用 n 个 21 的小矩形无重叠地覆盖一个 2*n 的大矩形，总共有多少种方法？

空间限制：C/C++ 64M，其他语言 128M
比如 n=3 时，2*3 的矩形块有 3 种覆盖方法。

先找一下规律，2n，因为一边的定长一定是 2，先想到的是动态规划。
21，1 种；2*2，2 种。
f(n) = f(n-1) + f(n-2);
也是一个斐波那契数列，没什么好说的，主要是要知道还有一种方法，用数组的方法，牺牲空间换取时间。

# 变态跳台阶 一只青蛙一次可以跳上 1 级台阶，也可以跳上 2 级…… 它也可以跳上 n 级。求该青蛙跳上一个 n 级的台阶总共有多少种跳法。

这次还是用动态规划的思想。
$f(n) = f(n-1) + f(n-2) +f(n-3) + ... + f(2) + f(1) $
提供一些比较好的思路：

  1. 用数组 vector，进行累加，不用重复。
  2. $f (n) = 2f (n-1) $ ，直接左移 (×2)。

# 二维数组中的查找 在一个二维数组中（每个一维数组的长度相同），每一行都按照从左到右递增的顺序排序，每一列都按照从上到下递增的顺序排序。请完成一个函数，输入这样的一个二维数组和一个整数，判断数组中是否含有该整数。

类二分法思想，首先要确定信标点在什么地方，肯定不可能是 [x][l] 为信标点，因为行末虽然更大，但是高行处也会有更小的数。
既然是二维数组，那么可以选择 [x][x] 作为信标。
最好的地方是 [x/√2] 的位置，大概 [2x/3]。不对，思路有错误。
要确定区间，其实不能是 [0][0]~[x][y] 区间，而应该是排除法，进行取反是绝对可以排除的。
用 count () 计算数组大小。

移动比较思路，从左下角开始，$t > $x 时，右移； $t < $x 时，上移。向三个角移动，可以先用 00 和 nn 比较确定是否存在，那么就只剩下一个角，0n，比较完之后则可。
在这个思路下，我们可以直接从 [n/2][n/2] 开始，移动到任何一个临角 (n0 或 0n) 时结束。

# 输入一个整数，输出该数 32 位二进制表示中 1 的个数。其中负数用补码表示。

模 2 并移位 ， 移位异或 都是 32 次。

val 和 (val - 1) 进行与操作。赋值给 val，直到 val 为 0。
空间复杂度直接就是 O (n)，n 为 1 的个数。

怎么理解？
val - 1.
10000 - 1 = 1111
10001 - 1 = 10000

一种理解是，利用与的特性。反正 val 和 任何数取与，得到的数中包含的 1 ，都不会超过 val。那不如就是找一个方法，消除 val 中的 1。并且消除 1 的个数还是一个一个消除的，那么就得到了本题结果。而一个一个消除的方法，就是取 d = 1，这样就跳过了一段 0 很多的情况。关于 d 取其他数，可行性未知，有可能会适得其反。

这是逆向思维，但是顺势思维未有多想。
