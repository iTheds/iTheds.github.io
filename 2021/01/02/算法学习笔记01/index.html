程序 = 算法 + 数据结构

# 树和链表

# 重建二叉树 输入某二叉树的前序遍历和中序遍历的结果，请重建出该二叉树。假设输入的前序遍历和中序遍历的结果中都不含重复的数字。例如输入前序遍历序列 {1,2,4,7,3,5,6,8} 和中序遍历序列 {4,7,2,1,5,3,8,6}，则重建二叉树并返回。

前序和中序遍历，指的是根节点在前和中。
所以主要的思想是截断，分块进行。

前序遍历中第一位 1，是根节点，中序遍历分块：{4,7,2}, {1} ,{5,3,8,6}。
然后再看前序 {2,4,7,3,5,6,8}，是 2，
left 分 {4,7}, {2} -> {4} ,{7}。
right 分 {5}, {3} ,{8,6} -> {8}, {6} 。
所以确定要用到递归。

# 旋转数组的最小数字 局部有序寻找最小值。

把一个数组最开始的若干个元素搬到数组的末尾，我们称之为数组的旋转。
输入一个非递减排序的数组的一个旋转，输出旋转数组的最小元素。
NOTE：给出的所有元素都大于 0，若数组大小为 0，请返回 0。

sync_with_stdio 这个函数是一个 “是否兼容 stdio” 的开关，C++ 为了兼容 C，保证程序在使用了 std::printf 和 std::cout 的时候不发生混乱，将输出流绑到了一起。

cin.tie()
cin.tie (NULL); 只解除的是 C++ 运行库层面的对数据传输的绑定。
STDIN 和 STDOUT 应该在更底层的操作系统层面有绑定，没有解除
可以通过 tie (0)（0 表示 NULL）
来解除 cin 与 cout 的绑定，进一步加快执行效率。

# NC78 反转列表：输入一个链表，反转链表后，输出新链表的表头。(1.4)

# 二维数组中的查找 在一个二维数组中（每个一维数组的长度相同），每一行都按照从左到右递增的顺序排序，每一列都按照从上到下递增的顺序排序。请完成一个函数，输入这样的一个二维数组和一个整数，判断数组中是否含有该整数。

类二分法思想，首先要确定信标点在什么地方，肯定不可能是 [x][l] 为信标点，因为行末虽然更大，但是高行处也会有更小的数。
既然是二维数组，那么可以选择 [x][x] 作为信标。
最好的地方是 [x/√2] 的位置，大概 [2x/3]。不对，思路有错误。
要确定区间，其实不能是 [0][0]~[x][y] 区间，而应该是排除法，进行取反是绝对可以排除的。
用 count () 计算数组大小。

移动比较思路，从左下角开始，$t > $x 时，右移； $t < $x 时，上移。向三个角移动，可以先用 00 和 nn 比较确定是否存在，那么就只剩下一个角，0n，比较完之后则可。
在这个思路下，我们可以直接从 [n/2][n/2] 开始，移动到任何一个临角 (n0 或 0n) 时结束。

# 替换空格 请实现一个函数，将一个字符串中的每个空格替换成 “%20”。例如，当字符串为 We Are Happy. 则经过替换之后的字符串为 We%20Are%20Happy。

考察替换函数 str_replace ();

如果是使用 C++ 实现，那么要注意从前往后和从后往前的区别。移动次数要数从后往前更少。
